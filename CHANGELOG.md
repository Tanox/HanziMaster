# Changelog

## [0.9.1]

### Added
- **UI Branding**: 现已在应用 Header 和主标题中直接显示版本号，方便用户识别。

### Changed
- **Version Alignment**: 同步了 `package.json`、`metadata.json` 和核心组件的版本号。

## [0.9.0]

### Added
- **Pinyin Typography Overhaul**: 引入专门的 `.font-pinyin` 字体栈，强制拼音使用 `Inter` 及系统默认无衬线字体，彻底解决音标（变音符号）在某些环境下位移或显示为“补丁”的问题。
