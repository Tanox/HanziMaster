# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.5.4] - 2025-05-24

### Changed
- **Visual Branding**: Redesigned all application icons (favicon, icon, maskable) with a traditional "Chinese Seal" aesthetic.
- **Color Palette**: Updated brand vermilion to `#bc221b` for better depth and reduced eye strain.
- **PWA Enhancement**: Optimized maskable icon safe zones for better cross-platform compatibility.
