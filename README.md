# 汉字大师 HanziMaster

汉字大师是一款现代化的汉字学习应用，通过 AI 技术帮助您掌握汉字的笔画、结构和书写美感。

## 主要功能

- **智能学习辅导** - AI 实时分析您的书写，提供个性化反馈
- **12 个基础汉字** - 从简单的常用字开始，循序渐进
- **多种学习模式** - 包含学习、练习、测验三种模式
- **进度追踪** - 记录学习历史，了解学习成效
- **深色/浅色模式** - 保护眼睛，适应不同环境
- **多语言界面** - 支持 11 种语言切换

## 支持的语言

简体中文、繁体中文、英语、西班牙语、阿拉伯语、法语、葡萄牙语、德语、日语、韩语、俄语

## 快速开始

### 环境要求

- Node.js >= 20.11.0
- npm 或 yarn

### 安装步骤

1. 克隆项目
   ```bash
   git clone <repository-url>
   cd <project-directory>
   ```

2. 安装依赖
   ```bash
   npm install
   ```

3. 启动应用
   ```bash
   npm run dev
   ```

4. 打开浏览器访问 `http://localhost:3000`

### AI 功能配置（可选）

如果需要 AI 书写分析功能：

1. 复制环境变量模板
   ```bash
   cp .env.example .env
   ```

2. 编辑 `.env` 文件，添加您的 Google Gemini API 密钥：
   ```
   GEMINI_API_KEY=您的密钥
   ```

## 页面介绍

### 首页

展示应用特色功能介绍，可快速访问学习资源。

### 学习页

- 选择要学习的汉字
- 查看汉字的拼音、笔画、含义
- 听标准发音
- 了解字源文化

### 练习页

- **书写练习** - 在方格中练习书写
- **记忆测验** - 测试对汉字的掌握程度
- **学习统计** - 查看本周学习进度

### Vercel
