# 汉字大师 HanziMaster

汉字大师是一款现代化的汉字学习应用，通过 AI 技术帮助您掌握汉字的笔画、结构和书写美感。

## 主要功能

- **智能学习辅导** - AI 实时分析您的书写，提供个性化反馈
- **丰富汉字库** - 包含常用汉字、词源文化、自适应学习
- **多种学习模式** - 包含学习、练习、测验三种模式
- **进度追踪** - 记录学习历史，了解学习成效
- **成就系统** - 收集徽章，激励持续学习
- **深色/浅色模式** - 保护眼睛，适应不同环境
- **多语言界面** - 支持 11 种语言切换

## 支持的语言

简体中文、繁体中文、英语、西班牙语、阿拉伯语、法语、葡萄牙语、德语、日语、韩语、俄语

## 技术特性

- **shadcn/ui** - 现代化 UI 组件库
- **书写画布** - Canvas API 实现真实书写交互
- **状态管理** - Zustand 实现进度持久化
- **数据缓存** - TanStack React Query 5
- **AI 集成** - Google Gemini API 支持
- **单元测试** - Jest + Testing Library

## 快速开始

### 环境要求

- Node.js >= 20.11.0
- npm 或 yarn

### 安装步骤

1. 克隆项目
   ```bash
   git clone <repository-url>
   cd <project-directory>
   ```

2. 安装依赖
   ```bash
   npm install
   ```

3. 启动应用
   ```bash
   npm run dev
   ```

4. 打开浏览器访问 `http://localhost:3000`

### AI 功能配置（可选）

如果需要 AI 书写分析功能：

1. 复制环境变量模板
   ```bash
   cp .env.example .env
   ```

2. 编辑 `.env` 文件，添加您的 Google Gemini API 密钥：
   ```
   GEMINI_API_KEY=您的密钥
   ```

## 页面介绍

### 首页

展示应用特色功能介绍，可快速访问学习资源，包含 Hero 字符轮播、学习流程四步、书写画布预览等。

### 学习页

- 选择要学习的汉字
- 查看汉字的拼音、笔画、含义
- 听标准发音
- 了解字源文化

### 练习页

- **书写练习** - 在方格中练习书写
- **记忆测验** - 测试对汉字的掌握程度
- **学习统计** - 查看本周学习进度
