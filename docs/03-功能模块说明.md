## 2. 核心功能模块

### 2.1 笔顺学习模块 (Stroke Learning)

**功能描述**: 核心学习模块，提供笔顺动画观看和书写练习功能

**主要组件**:
- `StrokeViewer.tsx`: 笔顺动画观看器
- `Controls.tsx`: 播放控制按钮
- `ViewerSection.tsx`: 观看模式主区域

**核心功能**:
1. **笔顺动画**
   - SVG 矢量渲染
   - 平滑的笔画流动动画
   - 支持 0.5x - 2x 速度调节
   - 播放、暂停、重置、重复

2. **练习书写**
   - Canvas 绘制支持
   - 实时几何评分
   - 笔画智能反馈
   - 多感官反馈（视觉、听觉、触觉）

**业务流程图**:
```
开始 → 选择汉字 → [观看模式] → 播放动画 → 调整速度
                                    ↓
                              [练习模式] → 书写第一笔 → 获得反馈 → 继续书写 → 完成练习
                                    ↓
                              继续下一个字/随机推荐
```

**关键逻辑**:
- **几何评分算法**: 计算用户轨迹与目标轨迹的欧氏距离和余弦相似度
- **三级降级策略**: 优先使用本地缓存，失败时使用 CDN，最后使用 AI
- **状态管理**: 通过 `useInteractionState` 管理瞬时 UI 状态

### 2.2 AI 解析模块 (AI Analysis)

**功能描述**: 利用 Google Gemini AI 提供深度汉字解析

**主要组件**:
- `CharacterDisplay.tsx`: 单字解析展示
- `IdiomDisplay.tsx`: 成语解析展示
- `AnalysisPanel.tsx`: 分析面板容器
- 信息卡片:
  - `StructureCard.tsx`: 结构解析
  - `StrokeCountCard.tsx`: 笔画数
  - `EtymologyCard.tsx`: 字源故事
  - `MnemonicCard.tsx`: 记忆口诀
  - `ExampleWordsCard.tsx`: 常用词组
  - `OfflineStateCard.tsx`: 离线状态提示

**核心功能**:
1. **单字解析**
   - 拼音标注
   - 释义
   - 部首
   - 笔画数
   - 字源故事
   - 记忆口诀
   - 常用词组

2. **成语解析**
   - 成语拼音
   - 释义
   - 典故/出处
   - 造句/用法示例

3. **离线降级**
   - 智能检测网络状态
   - 提供简化的离线信息
   - 自动切换降级策略

**业务流程图**:
```
搜索汉字/成语 → 检查网络状态 → [在线] → 调用 Gemini API → 解析 JSON → 展示解析
                                          ↓
                                    [离线] → 显示离线状态 → 提供基础信息
```

**关键逻辑**:
- **三级降级策略**: L1 本地缓存 → L2 CDN → L3 AI Engine
- **JSON 清理**: 健壮地从 Gemini 返回的文本中抽取 JSON（处理 markdown 包裹等情况）
- **Schema 验证**: 使用 Zod 验证 AI 返回的结构化数据

### 2.6 分享功能模块 (Share)

**功能描述**: 将汉字笔顺生成为图片分享

**主要组件**:
- `ShareButton.tsx`: 分享按钮
- `ShareImageButton.tsx`: 图片分享按钮
- `PromoShareButton.tsx`: 推广分享按钮

**核心功能**:
- 分享为图片
- 复制链接
- 推广文案生成
- 图片生成（Canvas 渲染）

**关键逻辑**:
- **纯前端图像生成**: 使用 HTML5 Canvas API，无需后端服务
- **字体加载**: 监听 `document.fonts.ready` 事件，确保字体已加载
- **主题适配**: 自动读取当前 DOM 的 `dark` class，生成对应配色的图片

## 3. 核心 Hooks

### 3.1 useAppController
**职责**: 应用的"大脑"，全局状态协调者

**主要功能**:
- URL 参数解析
- 全局 Modal 状态管理
- 协调各子 Hook 间的交互
- 网络状态监听
- 主题管理

### 3.2 useContentFetcher
**职责**: 数据获取与缓存

**主要功能**:
- 获取 HanziData（笔顺数据）
- 获取 CharacterAnalysis（AI 解析）
- 获取 IdiomAnalysis（成语解析）
- 智能缓存管理
- 三级降级策略

### 3.3 useInteractionState
**职责**: 瞬时 UI 状态管理

**主要功能**:
- 播放/暂停/重置
- 模式切换（Watch/Practice）
- 动画进度
- 速度调节

### 3.4 useUserProgress
**职责**: 用户学习进度管理

**主要功能**:
- 学习历史记录
- 已学清单管理
- LocalStorage 同步
- 学习统计

### 3.5 useDataSync
**职责**: 离线数据同步

**主要功能**:
- 笔顺库下载
- 词典下载
- 缓存审计
- 数据健康度检查

## 4. 核心服务

### 4.1 geminiService
**职责**: Google Gemini AI 交互

**主要功能**:
- 文本生成（汉字解析）
- TTS 生成
- 离线降级
- 错误处理
- 请求重试

**关键逻辑**:
- 必须使用命名参数初始化客户端
- 支持强制离线模式
- 智能检测网络状态
- 健壮的 JSON 解析

### 4.2 hanziService
**职责**: 汉字笔顺数据服务

**主要功能**:
- 笔顺数据获取
- 本地缓存管理
- CDN 回退
- 数据验证

### 4.3 ttsService
**职责**: 文本转语音服务

**主要功能**:
- Gemini TTS 调用
- 浏览器原生 TTS 回退
- 音频缓存
- 音频播放控制

### 4.4 soundService
**职责**: 音效服务

**主要功能**:
- 练习音效播放
- 成功/失败反馈音
- 音量控制
