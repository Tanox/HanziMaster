# 用户操作指南

---

## 1. 角色与权限

HanziMaster 是一个单用户应用，所有功能对用户开放：

| 角色 | 权限 | 说明 |
|------|------|------|
| **普通用户** | 完整功能 | 所有功能可自由使用 |

**注意**: 应用没有登录系统，所有数据仅存储在浏览器本地。

---

## 2. 快速入门

### 2.1 首次使用

1. 打开应用后，会看到欢迎屏幕
2. 跟随引导了解主要功能：
   - 搜索汉字
   - 观看笔顺动画
   - 练习书写
3. 点击"开始使用"进入主界面

### 2.2 基本流程

```
搜索汉字 → 观看笔顺 → 练习书写 → AI 解析 → 分享成果
```

---

## 3. 核心功能操作

### 3.1 搜索汉字

**功能描述**: 搜索想要学习的汉字或成语

**操作步骤**:
1. 在顶部搜索框中输入汉字或成语
2. 点击搜索按钮或按回车键
3. 系统会显示搜索结果

**快捷键**:
- `Enter`: 执行搜索
- `Esc`: 清空搜索

**提示**: 可以使用下方的"随机推荐"快速选择汉字

---

### 3.2 观看笔顺动画

**功能描述**: 观看汉字的正确书写顺序

**操作步骤**:
1. 搜索并选择一个汉字
2. 在"观看"模式下，点击播放按钮
3. 观察笔顺动画
4. 可以调整播放速度（0.5x - 2x）

**控制按钮**:
- ▶️ 播放 / ⏸️ 暂停
- ⏹️ 重置
- 🔄 重复播放

**操作流程图**:
```
选择汉字 → 进入观看模式 → 点击播放 → 观看动画 → 调整速度
```

---

### 3.3 练习书写

**功能描述**: 在画布上练习书写，获得即时反馈

**操作步骤**:
1. 点击"练习"切换到练习模式
2. 按照提示在画布上书写
3. 系统会给出即时评分和反馈
4. 完成后可以继续下一个字

**评分等级**:
- 神品 (95-100分)
- 妙品 (85-94分)
- 能品 (75-84分)
- 须努力 (<75分)

**提示**: 
- 确保落笔点和起笔方向正确
- 保持笔画流畅

**操作流程图**:
```
切换到练习模式 → 书写第一笔 → 获得反馈 → 继续书写 → 完成练习
```

---

### 3.4 AI 解析

**功能描述**: 查看汉字的详细解析，包括字源、助记、例词等

**操作步骤**:
1. 搜索并选择一个汉字
2. 查看右侧的分析面板
3. 可以展开/折叠不同的信息卡片：
   - 结构解析
   - 笔画数
   - 字源故事
   - 记忆口诀
   - 常用词组

**提示**: 
- AI 解析需要网络连接和 API Key
- 离线模式下会显示简化信息

---

### 3.5 发音功能

**功能描述**: 听汉字的正确发音

**操作步骤**:
1. 点击分析面板上的发音按钮（喇叭图标）
2. 系统会播放发音
3. 可以重复播放

**说明**: 
- 在线模式使用 Gemini TTS
- 离线模式使用浏览器原生语音合成

---

### 3.6 分享功能

**功能描述**: 将汉字笔顺生成为图片分享

**操作步骤**:
1. 点击分享按钮
2. 选择分享方式：
   - 分享为图片
   - 复制链接
3. 保存或分享生成的内容

---

## 4. 设置面板

### 4.1 界面设置

| 设置项 | 说明 | 选项 |
|--------|------|------|
| **网格样式** | 书写网格 | 米字格 / 田字格 / 无 |
| **显示轮廓** | 显示汉字轮廓 | 开 / 关 |
| **主题** | 界面主题 | 浅色 / 深色 / 跟随系统 |
| **语言** | 界面语言 | 15+ 种语言 |

### 4.2 学习设置

| 设置项 | 说明 | 选项 |
|--------|------|------|
| **自动播放** | 搜索后自动播放动画 | 开 / 关 |
| **连续模式** | 完成后自动进入下一个 | 开 / 关 |
| **离线模式** | 强制使用离线功能 | 开 / 关 |
| **音效** | 练习音效 | 开 / 关 |

### 4.3 内容设置

可以选择显示/隐藏的信息卡片：
- 结构解析
- 字源故事
- 记忆口诀
- 常用词组

### 4.4 API Key 设置

**操作步骤**:
1. 打开设置面板
2. 进入"API Key"部分
3. 输入您的 Google Gemini API Key
4. 点击保存

**获取 API Key**: 访问 [Google AI Studio](https://aistudio.google.com/)

### 4.5 数据管理

**重置应用**:
1. 打开设置面板
2. 进入"危险区域"
3. 点击"重置应用"
4. 确认操作

**警告**: 此操作会清除所有本地数据！

---

## 5. 历史记录

### 5.1 查看历史

**操作步骤**:
1. 点击历史记录按钮
2. 查看最近的练习记录
3. 点击某个记录可以重新练习

### 5.2 清除历史

**操作步骤**:
1. 打开历史记录面板
2. 点击"清除历史"
3. 确认操作

---

## 6. PWA 安装

### 6.1 安装到桌面

**Chrome/Edge**:
1. 访问应用
2. 点击地址栏右侧的安装图标
3. 点击"安装"

**移动端**:
1. 访问应用
2. 使用浏览器的"添加到主屏幕"功能
3. 按提示操作

---

## 7. 离线使用

### 7.1 离线功能

即使没有网络连接，以下功能仍然可用：
- ✅ 笔顺动画
- ✅ 练习书写
- ✅ 基本拼音显示
- ✅ 历史记录

### 7.2 离线限制

需要网络的功能：
- ❌ AI 深度解析
- ❌ Gemini TTS 发音
- ❌ 分享（需要网络上传）

## 8. 快捷键

| 快捷键 | 功能 |
|--------|------|
| `/` 或 `Ctrl+K` | 聚焦搜索框 |
