# 常见问题与故障排除指南

---

## 1. 常见问题 (FAQ)

### 1.1 基础问题

**Q: HanziMaster 是什么？**
A: HanziMaster 是一个融合传统书法美学与现代 AI 智慧的沉浸式汉字教学平台，通过 Watch → Trace → Write 的 Scaffolding 教学法帮助用户掌握汉字书写。

**Q: 这个应用需要联网吗？**
A: 核心功能（笔顺动画、练习书写）完全离线可用。AI 深度解析、Gemini TTS 发音等功能需要网络连接。

**Q: 我的数据会被上传吗？**
A: 不会。HanziMaster 采用零知识（Zero-Knowledge）设计，所有数据（练习历史、设置、API Key）仅存储在您的浏览器本地。

**Q: 支持哪些语言？**
A: 界面支持 15+ 种语言，包括简体中文、繁体中文、英语、西班牙语、德语、日语等。

---

### 1.2 使用问题

**Q: 汉字无法加载怎么办？**
A: 
1. 检查 `public/hanzi-data/` 目录是否存在
2. 手动运行数据同步：`npm run copy-data`
3. 刷新页面重试

**Q: 练习时没有响应？**
A: 
1. 确认您在"练习"模式（不是"观看"模式）
2. 检查是否有其他弹窗阻止了操作
3. 尝试刷新页面
4. 检查浏览器是否支持触摸/鼠标事件

**Q: AI 解析不显示？**
A: 
1. 检查是否有网络连接
2. 确认已配置 API Key（设置 → API Key）
3. 检查是否在离线模式（设置 → 学习设置）
4. 查看浏览器控制台（F12）是否有错误

**Q: 发音功能不工作？**
A: 
1. 在线模式：检查 API Key 和网络连接
2. 离线模式：确认浏览器支持 `speechSynthesis`
3. 检查浏览器音量设置
4. 尝试刷新页面

**Q: 如何获取 Google Gemini API Key？**
A: 
1. 访问 [Google AI Studio](https://aistudio.google.com/)
2. 登录 Google 账号
3. 创建新的 API Key
4. 将 Key 复制到 HanziMaster 设置中

**Q: API Key 安全吗？**
A: 是的，API Key 仅存储在您的浏览器本地，不会上传到任何服务器。

---

### 1.3 设置与配置

**Q: 如何切换主题？**
A: 设置 → 外观 → 主题（浅色/深色/跟随系统）

**Q: 如何更改界面语言？**
A: 设置 → 外观 → 语言

**Q: 可以关闭某些信息卡片吗？**
A: 可以，设置 → 内容 → 选择要显示/隐藏的卡片

**Q: 如何重置所有数据？**
A: 设置 → 危险区域 → 重置应用（警告：会清除所有本地数据！）

---

### 1.4 PWA 相关

**Q: 如何安装到桌面？**
A: 
- **Chrome/Edge**: 点击地址栏右侧的安装图标
- **移动端**: 使用浏览器的"添加到主屏幕"功能

**Q: PWA 更新后没有变化？**
A: 
1. 关闭所有应用窗口
2. 重新打开应用
3. 如果有更新提示，点击刷新

**Q: 离线时 PWA 还能用吗？**
A: 可以，核心功能完全支持离线使用。

## 2. 故障排除

### 2.1 页面加载问题

**症状**: 页面白屏或无法加载

**排查步骤**:
1. 检查浏览器控制台（F12）的错误信息
2. 确认 `npm install` 已完成
3. 检查 `public/hanzi-data/` 目录是否存在
4. 尝试 `npm run copy-data` 手动同步数据
5. 清除浏览器缓存并刷新

**解决方案**:
```bash
# 清理并重新安装
rm -rf node_modules .next
npm install
npm run copy-data
npm run dev
```

---

### 2.2 笔顺数据问题

**症状**: 汉字无法显示或动画异常

**排查步骤**:
1. 检查网络连接（首次加载需要下载数据）
2. 确认 `public/hanzi-data/` 目录包含 JSON 文件
3. 检查浏览器控制台是否有 404 错误
4. 尝试不同的汉字（可能是生僻字）

**解决方案**:
```bash
# 手动同步数据
npm run copy-data
```

---

### 2.3 AI 服务问题

**症状**: AI 解析失败或超时

**排查步骤**:
1. 确认有网络连接
2. 检查 API Key 是否正确配置
3. 查看 API Key 是否有配额限制
4. 检查浏览器控制台的错误信息
5. 尝试切换到离线模式

**常见错误码**:
- `400 Bad Request`: 请求格式错误
- `401 Unauthorized`: API Key 无效
- `429 Too Many Requests`: 超过配额限制
- `5xx Server Error`: 服务端问题，稍后重试

**解决方案**:
1. 确认 API Key 正确
2. 检查 Google Cloud 控制台的配额
3. 尝试使用离线模式
4. 稍后重试

---

### 2.4 练习书写问题

**症状**: 画布无法书写或评分异常

**排查步骤**:
1. 确认在"练习"模式
2. 检查是否有其他元素遮挡了画布
3. 尝试不同的浏览器
4. 检查触摸/鼠标事件是否被阻止
5. 确认画笔设置正确

**解决方案**:
1. 刷新页面
2. 尝试使用不同的浏览器（推荐 Chrome/Edge）
3. 清除浏览器缓存
4. 检查是否有浏览器扩展干扰

---

### 2.5 存储问题

**症状**: 设置或历史记录丢失

**排查步骤**:
1. 检查浏览器是否启用了 LocalStorage
2. 确认没有使用无痕/隐私模式
3. 检查浏览器存储空间是否已满
4. 确认没有手动重置应用

**解决方案**:
1. 检查浏览器设置，确保 LocalStorage 启用
2. 使用普通模式（非无痕）
3. 清理浏览器存储空间
4. 定期导出重要数据（如有）

---

### 2.6 性能问题

**症状**: 应用卡顿或动画不流畅

**排查步骤**:
1. 检查设备性能
2. 关闭其他浏览器标签页
3. 尝试减少同时打开的应用
4. 检查浏览器是否需要更新

**优化建议**:
1. 使用现代浏览器（Chrome/Edge/Firefox）
2. 关闭不必要的浏览器扩展
3. 确保设备有足够的内存
4. 定期清理浏览器缓存

## 3. 已知问题与限制

### 3.1 浏览器兼容性

| 浏览器 | 版本 | 支持状态 |
|--------|------|----------|
| Chrome | 90+ | ✅ 完全支持 |
| Edge | 90+ | ✅ 完全支持 |
| Firefox | 88+ | ✅ 完全支持 |
| Safari | 14+ | ⚠️ 部分支持（TTS 可能有问题） |
| IE | 任何 | ❌ 不支持 |

### 3.2 功能限制

- **生僻字**: 部分生僻字可能没有笔顺数据
- **离线 AI**: 离线模式下 AI 解析功能受限
- **存储限制**: LocalStorage 有大小限制（通常 5-10MB）

### 3.3 移动端限制

- **小屏幕**: 部分功能在超小屏幕上可能体验不佳
- **触摸精度**: 移动设备触摸精度可能影响评分
