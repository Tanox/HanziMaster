
# 08. 用户旅程 (User Journey)

## 1. 典型场景 A：日常偶遇
*   **触发**: 用户在报纸上看到“墨”字，不知其写。
*   **行动**: 打开 HanziMaster -> 搜索“墨” -> 观察笔顺动画。
*   **获得**: 发现原来是“土”在下，并听到了正确发音。

## 2. 典型场景 B：HSK 深度练习
*   **触发**: 用户准备 HSK 4 级考试，需要记忆“关键”一词。
*   **行动**: 输入“关键” -> 利用 IdiomNavigator 在两字间切换练习 -> 查看 AI 生成的记忆口诀。
*   **成就**: 在本地历史记录中看到该词已点亮。

## 3. 典型场景 C：成语典故探索
*   **触发**: 学习“守株待兔”。
*   **行动**: 搜索全词 -> 进入成语模式 -> 阅读 AI 生成的韩非子原典故事。
*   **情感反馈**: 对汉字的起源产生敬畏感，通过社交分享卡片向朋友炫耀。
