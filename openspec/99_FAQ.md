# 99. 常见问题与故障排除 (FAQ & Troubleshooting)

**项目**: HanziMaster (汉字大师)
**版本**: v1.4.0
**状态**: 现行规范

## 1. 常见问题 (FAQ)

### Q1: 为什么有些汉字没有笔顺动画？
**A**: HanziMaster 使用开源的 `hanzi-writer-data` 数据库，目前覆盖了约 9000 个常用汉字。对于极生僻字或异体字，可能暂时无法显示笔顺。我们会持续更新字库。

### Q2: 为什么 AI 解析有时候加载很慢？
**A**: AI 解析功能依赖于 Google Gemini API，响应速度受网络状况和 API 负载影响。如果长时间无响应，请尝试刷新页面或检查网络连接。

### Q3: 离线模式下还能使用 AI 功能吗？
**A**: 不能。离线模式仅支持本地缓存的汉字笔顺动画和基础释义。AI 解析（字源、记忆口诀等）需要联网才能生成。

### Q4: 如何清除历史记录？
**A**: 在首页向下滚动至“最近学习”列表，点击右侧的垃圾桶图标即可清空所有历史记录。

### Q5: 支持繁体字吗？
**A**: 支持。在设置中可以将目标学习语言切换为“繁体中文”，系统会自动调整笔顺和释义。

## 2. 故障排除指南 (Troubleshooting)

### 2.1 笔顺动画不显示
*   **现象**: 汉字详情页加载后，画布区域空白或只显示加载图标。
*   **原因**:
    1.  网络问题导致 SVG 数据下载失败。
    2.  浏览器缓存异常。
    3.  该汉字不在字库中。
*   **解决步骤**:
    1.  刷新页面 (F5 / Cmd+R)。
    2.  清除浏览器缓存 (Ctrl+Shift+R / Cmd+Shift+R)。
    3.  检查控制台 (F12) 是否有 404 错误。如果是 404，说明该字暂不支持。

### 2.2 AI 解析报错 "Service Unavailable"
*   **现象**: AI 助教面板显示红色错误提示。
*   **原因**:
    1.  API Key 配额耗尽或无效。
    2.  网络连接不稳定。
    3.  Gemini 服务端故障。
*   **解决步骤**:
    1.  检查 `.env.local` 中的 `NEXT_PUBLIC_GEMINI_API_KEY` 是否正确配置。
    2.  尝试稍后重试。
    3.  如果是自部署版本，请检查 Vercel 后台的环境变量设置。

### 2.3 无法发音 (TTS)
*   **现象**: 点击朗读按钮无声音。
*   **原因**:
    1.  浏览器不支持 Web Speech API。
    2.  系统静音或音量过低。
    3.  网络问题导致 Google Cloud TTS 无法连接。
*   **解决步骤**:
    1.  确保使用 Chrome, Edge, Safari 等现代浏览器。
    2.  检查系统音量设置。
    3.  尝试切换网络环境。

### 2.4 移动端书写不灵敏
*   **现象**: 手指划过屏幕但笔迹断断续续。
*   **原因**:
    1.  屏幕贴膜影响触控。
    2.  浏览器缩放比例不正确。
    3.  手掌误触。
*   **解决步骤**:
    1.  尝试清洁屏幕。
    2.  确保浏览器缩放比例为 100%。
    3.  尝试使用触控笔书写。
