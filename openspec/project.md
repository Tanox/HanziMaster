# HanziMaster 项目总纲

**版本**: v0.8.6
**状态**: 稳定运行中 (Scoring Version)
**愿景**: 融合传统书法美学与 AI 智慧的**沉浸式汉字教学平台**。

## 1. 核心价值 (Core Values)
*   **教学为本 (Pedagogy First)**: 提供“观察 -> 临摹 -> 评估”的学习闭环。
*   **纸墨体验 (Paper & Ink)**: 还原宣纸与朱砂的真实交互感。
*   **智能辅助 (AI Powered)**: v0.8.0 智能评分算法 + Gemini 字源解析。
*   **隐私优先 (Privacy First)**: BYOK 模式，数据本地化。

## 2. 核心功能模块 (v0.8.6)
1.  **智能评分仪 (AI Evaluator)**: 基于几何向量的全字评分系统。
2.  **笔顺教学仪 (Stroke Viewer)**: 矢量流体动画演示。
3.  **临摹练字板 (Practice Pad)**: 实时校验与虚影救生圈。
4.  **离线书房 (PWA Engine)**: 高性能离线同步与三级加载机制。

## 3. 技术栈摘要
*   React 18.3.1, Vite, TypeScript.
*   @google/genai (v0.2.0+).
*   Workbox PWA.

## 4. 里程碑 (Roadmap)
*   **v0.8.x**: 完善评估逻辑与离线体验（当前阶段）。
*   **v0.9.0**: (规划) 引入 HSK 级联课程体系。
*   **v1.0.0**: (规划) 正式版发布。
