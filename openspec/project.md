# HanziMaster 项目总纲

**版本**: v0.9.1
**状态**: 品牌化完成 (Branding Version)
**愿景**: 融合传统书法美学与 AI 智慧的**沉浸式汉字教学平台**。

## 1. 核心价值 (Core Values)
*   **教学为本 (Pedagogy First)**: 提供“观察 -> 临摹 -> 评估”的学习闭环。
*   **纸墨体验 (Paper & Ink)**: 还原宣纸与朱砂的真实交互感。
*   **排版精度 (Typography)**: 专为拼音设计的字体渲染栈，确保音标无损显示。
*   **品牌识别 (Branding)**: 统一的版本管理与 UI 标识。

## 2. 核心功能模块 (v0.9.1)
1.  **智能评分仪 (AI Evaluator)**: 基于几何向量的全字评分系统。
2.  **拼音排版引擎 (Pinyin Typography)**: 解决音标偏移问题的专用 CSS 渲染方案。
3.  **临摹练字板 (Practice Pad)**: 实时校验与虚影救生圈。
4.  **离线书房 (PWA Engine)**: 高性能离线同步与三级加载机制。

## 3. 技术栈摘要
*   React 18.3.1, Vite, TypeScript.
*   @google/genai (v0.2.0+).
*   Workbox PWA (Offline-First).

## 4. 里程碑 (Roadmap)
*   **v0.8.x**: 完善评估逻辑与离线体验。
*   **v0.9.x**: (当前) 视觉品牌化、排版修复与 UI 优化。
*   **v1.0.0**: (规划) 引入 HSK 级联课程体系。
