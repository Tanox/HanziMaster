# HanziMaster 项目总纲

**版本**: v0.7.1
**状态**: 稳定迭代中
**愿景**: 融合传统书法美学与 AI 智慧的**沉浸式汉字教学平台**。

## 1. 核心价值 (Core Values)
*   **教学为本 (Pedagogy First)**: 一切功能服务于“学会写字”这一核心目标。我们构建了“观察 -> 临摹 -> 默写”的完整学习闭环，以巩固用户的视觉和肌肉记忆。
*   **纸墨体验 (Paper & Ink)**: 坚持 **Light Mode (亮色模式)** 为默认体验，通过噪点纹理和精心选择的配色，在数字设备上还原宣纸上书写的真实感与专注感，拒绝过度的数字化光污染。
*   **智能辅助 (AI Powered)**: 我们不让 AI 替代练习，而是利用 Gemini AI 解决“这个字为什么这么写”的认知难题，提供字源、记忆法及高质量发音支持，让学习更深入、更有趣。
*   **隐私优先 (Privacy First)**: 用户的所有学习数据和 API Key 均存储在本地，我们不设服务器，不收集用户数据，确保学习过程的私密性。

## 2. 核心功能模块 (v0.7.1)
1.  **笔顺教学仪 (Stroke Viewer)**: 高精度的矢量动画演示，支持分步暂停与多级速度调节。
2.  **临摹练字板 (Practice Pad)**: 带有实时几何校验的手写板，包含“虚影救生圈”辅助机制，确保用户在正确的路径上练习。
3.  **智能字源馆 (AI Etymology)**: Gemini AI 驱动的汉字文化解析、助记词生成和成语故事讲解。
4.  **离线书房 (Offline PWA)**: 完整的 PWA 支持，包含手动数据审计与离线数据包下载功能，保障随时随地学习。

## 3. 关键规范摘要
*   **默认主题**: Light Mode (`#fdfbf7`) + 宣纸纹理 (`bg-texture-paper`)。
*   **技术栈**: React 18, Vite, TypeScript, @google/genai (v0.2.0+)。
*   **数据源**: Hanzi Writer Data (笔顺), Google Gemini (解析), 本地缓存 (CacheStorage / LocalStorage)。
*   **开发原则**: 文档驱动开发 (Documentation-Driven Development)，`openspec/` 目录下的文档是项目的唯一真理来源。

## 4. 里程碑 (Roadmap)
*   **v0.7.x**: 稳定核心教学功能，完善文档体系。
*   **v0.8.0**: 引入智能评分系统和 HSK 课程体系。
*   **v0.9.0**: (规划) 引入用户手写轨迹回放和 AI 批改建议。
*   **v1.0.0**: (规划) 正式版发布，包含完整的 HSK 1-6 级课程。
