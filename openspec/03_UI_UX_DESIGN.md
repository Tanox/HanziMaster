# 03. UI/UX 设计规范

## 1. 视觉语言 (v0.6.0 更新)

### 3.7 虚影引导 (Ghosting Guide)
*   **颜色**: 使用 `vermilion-200` (浅朱砂)。
*   **动画**: 进入时采用 `fade-in` 动效 (0.3s)，并伴随轻微的 `pulse` 呼吸感。
*   **层级**: 位于网格线之上，用户手写轨迹之下。

### 3.8 部首面板 (Radical Panel)
*   **交互**: 采用侧边抽屉 (Drawer) 形式，不干扰主搜索框。
*   **排版**: 紧凑的网格布局，选中的部首以 `teal-600` 高亮。

## 4. 响应式适配
*   **折叠屏/大屏**: 在 iPad 等大屏设备上，部首面板可作为侧边栏常驻。
*   **触控优化**: PDF 生成按钮需具有明显的“打印”图标标识，位于练习成功弹窗的次要位置。
