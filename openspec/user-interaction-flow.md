# 中文字笔画顺序教学应用用户交互流程

## 1. 汉字选择阶段
1. **进入首页**：用户打开应用，看到首页界面。
2. **浏览推荐**：用户可以浏览首页的“常用字推荐”或“随机推荐”区域。
3. **主动搜索**：用户点击顶部的搜索框，输入想要学习的汉字（支持拼音或汉字输入）。
4. **确认选择**：用户点击推荐字或搜索结果中的汉字，进入该汉字的详情/学习页面。

## 2. 笔画演示阶段（观察模式）
1. **自动演示**：进入详情页后，系统默认自动播放一次完整的笔顺动画。
2. **控制播放**：
   - 用户点击“播放/暂停”按钮控制动画。
   - 用户拖动进度条查看特定笔画。
   - 用户点击“重播”按钮重新开始演示。
3. **查看详情**：
   - 用户点击“分步”按钮，可以一步一步查看每个笔画的写法。
   - 界面同步显示当前笔画的名称（如“横”、“竖钩”）和顺序编号。

## 3. 临摹练习阶段（练习模式）
1. **切换模式**：用户点击页面上的“练习”或“写一写”按钮，进入临摹模式。
2. **引导提示**：
   - 画面显示汉字的浅色轮廓。
   - 第一个笔画的起笔点出现闪烁提示。
3. **书写交互**：
   - 用户在屏幕上用手指或鼠标沿着提示轨迹书写。
   - **正确**：如果轨迹和方向正确，笔画被填实，并播放正确音效，提示点移动到下一笔。
   - **错误**：如果轨迹偏离或方向错误，笔画变红并抖动，播放错误音效，提示用户重写当前笔画。
4. **完成练习**：用户依次写完所有笔画。

## 4. 结果反馈阶段
1. **练习结算**：当最后一笔书写完成后，系统弹出结算界面。
2. **评分展示**：
   - 显示本次练习的综合评分（如 95分）和星级评价。
   - 显示准确度和流畅度的具体数值。
3. **回顾与改进**：
   - 界面上标记出用户写得不够好的笔画（如“第3笔 撇 写得太短”）。
   - 提供“回放我的轨迹”功能，对比标准动画和用户实际轨迹。
4. **后续操作**：
   - 点击“再写一次”重新练习当前字。
   - 点击“下一个”进入下一个推荐汉字的练习。
   - 点击“分享”将成绩分享给好友。
