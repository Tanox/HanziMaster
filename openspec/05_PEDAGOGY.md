
# 05. 教学大纲与学习算法

## 1. 核心教学法：脚手架理论 (Scaffolding)
产品遵循“观察 -> 临摹 -> 默写”的学习闭环，旨在通过逐步减少辅助来巩固用户的肌肉记忆和结构理解。
1.  **观察 (Watch Mode)**: 用户观看标准笔顺动画，建立视觉印象。
2.  **临摹 (Practice Mode - with Outline)**: 用户在灰色轮廓的引导下书写，关注笔画的走向和相对位置。
3.  **默写 (Practice Mode - without Outline)**: 用户关闭轮廓，仅依靠米字格进行书写，检验学习成果。

## 2. 课程内容体系：基于 HSK 3.0 标准
教学内容严格参照 **《国际中文教育中文水平等级标准》(HSK 3.0)**，确保学习的权威性和实用性。

### 内容覆盖现状
*   **笔顺数据**:
    *   通过 `hanzi-writer-data` 覆盖超过 9000个常用字，包含 HSK 1-6 级的全部汉字。
    *   用户可在设置中下载完整笔顺库，实现 100% 离线笔顺演示与练习。
*   **离线释义**:
    *   内置 `SIMPLE_DICTIONARY` 包含约 518 个 HSK 1-3 级核心汉字与词语的简明英文释义。
    *   用户可在设置中一键加载，作为离线或 AI 服务不可用时的兜底。
*   **AI 深度解析**:
    *   联网状态下，为所有可查询的汉字和成语提供动态的、多语言的深度解析（字源、助记、用法等）。

## 3. 学习算法规划：SRS (间隔重复系统)
*   **状态**: 规划中 (v1.1.x)
*   **算法**: 计划采用基于 SuperMemo 2 (SM-2) 算法的变体。
*   **模型**:
    ```typescript
    interface CharacterProgress {
      char: string;
      level: number;        // 掌握等级 (0-5)
      nextReview: number;   // 下次复习的时间戳
      interval: number;     // 复习间隔天数
    }
    ```
*   **目标**: 根据用户的练习得分 (`PracticeResult.score`) 和历史记录，智能安排复习计划，生成个性化的“每日复习”列表，从而科学高效地对抗遗忘曲线。
