# 01. 产品需求文档 (PRD)

**项目**: HanziMaster (汉字大师)
**版本**: 0.3.0
**状态**: 开发中 (Beta)

## 1. 执行摘要
HanziMaster 是一个基于 Web 的教育工具，旨在弥合传统死记硬背汉字与现代语言学理解之间的鸿沟。通过结合矢量笔画动画与生成式 AI (Google Gemini)，它为用户提供即时的视觉指导和深层的语义背景。

## 2. 用户画像
*   **初学者**: 需要知道如何书写“你好”而不像画画一样。需要标准的发音。
*   **学者型**: 想知道为什么“休”字是一个人靠在树上（会意字）。
*   **旅行者**: 需要一本在飞机上或信号差的地方也能可靠使用的字典。

## 3. 功能需求

### 3.1 核心学习功能 (支持离线)
*   **笔顺动画**:
    *   必须渲染 9000+ 个汉字的逐笔动画。
    *   控制功能：播放、暂停、重置、速度调节 (0.5x, 1x, 1.5x)。
    *   必须通过 PWA 预缓存，在完全无网络的情况下工作。
*   **交互模式**:
    *   **演示模式 (View Mode)**: 被动观看，支持 SVG 遮罩动画。
    *   **临摹模式 (Practice Mode)**: 用户在画布上书写，系统基于几何距离验证准确性并提供触觉/视觉反馈。
*   **原生 TTS (降级方案)**:
    *   离线时，使用 `window.speechSynthesis` 进行发音，自动匹配 `zh-CN` 或相关语音包。

### 3.2 AI 集成 (在线增强)
*   **语言学解析**:
    *   使用 Gemini 3 Flash 生成：拼音、部首、字源、记忆口诀和常用词汇。
    *   优雅降级：如果网络请求失败，显示基础本地数据或“网络不可用”状态，而不导致应用崩溃。
*   **神经 TTS**:
    *   在线时使用 Gemini 2.5 Flash TTS 提供自然、拟人化的语调。
    *   实现内存级缓存，避免重复请求同一字符。

### 3.3 探索与参与
*   **搜索**: 快速按字符查找，支持输入验证。
*   **随机推荐**:
    *   在界面底部显示 3-6 个随机汉字（响应式数量）。
    *   来源为本地存储的 5000 个常用汉字列表。
*   **多语言支持**: 界面和 AI 内容必须支持 12 种主要语言 (i18n)。

### 3.4 学习进度 (v0.3 已实现)
*   **本地历史记录**:
    *   自动记录最近练习的 20 个汉字。
    *   支持点击历史记录快速回看。
    *   数据存储于本地 (LocalStorage)，保护隐私。
*   **连续练习模式**:
    *   临摹完成后自动跳转到下一个随机汉字，提供沉浸式练习体验。

### 3.5 未来规划 (v0.4+)
*   **间隔重复系统 (SRS)**: 基于遗忘曲线自动安排复习时间 (算法: SM-2)。
*   **数据可视化**: 学习热力图 (Heatmap) 和统计看板。
*   **PWA 增强**: 实现后台同步 (Background Sync)。

## 4. 非功能性需求
*   **性能**: 首次内容绘制 (FCP) < 1.0s。可交互时间 (TTI) < 1.5s。
*   **可靠性**: 离线模式必须支持所有静态资源和已缓存的 API 调用。
*   **无障碍性 (A11y)**: 所有图标按钮需包含 ARIA 标签；播放器控件支持键盘导航。
*   **兼容性**: 支持现代浏览器 (Chrome, Safari, Edge, Firefox) 和移动操作系统 (iOS, Android)。

## 5. 路线图
*   **v0.1**: 基础笔画渲染。(已完成)
*   **v0.2**: AI 解析、TTS 和 PWA 离线支持。(已完成)
*   **v0.3**: 历史记录面板、随机推荐算法、多语言 UI 完善。(已完成 - 当前版本)
*   **v0.4**: 引入 IndexedDB 支持大量数据存储，实现 SRS 复习算法。
