## 2. 项目简介

HanziMaster 汉字大师是一个现代化的、基于人工智能的汉字学习平台。它采用**Apple 设计风格**：纯白背景为基调，配合毛玻璃（glassmorphism）效果，以 emerald 绿色作为强调色，圆角 24px 的卡片设计，柔和阴影层次，为用户提供愉悦且专注的汉字学习体验。

平台利用 Google Gemini AI 提供关于笔画顺序、平衡和美感的个性化反馈，帮助用户高效学习汉字。

## 3. 设计风格规范

### 3.1 核心设计原则

| 原则 | 描述 |
|-----|------|
| **极简主义** | 内容优先，去除一切不必要的装饰元素 |
| **层次分明** | 通过阴影、透明度创建清晰的视觉层次 |
| **柔和过渡** | 所有动画均使用缓动曲线，确保流畅自然 |
| **留白充足** |  generous whitespace 提升可读性与质感 |

### 3.2 视觉设计系统

| 设计元素 | 规范值 |
|---------|-------|
| **背景色** | `#ffffff`（纯白） |
| **强调色** | `#10b981`（emerald-500） |
| **强调色（深）** | `#059669`（emerald-600） |
| **主文本色** | `#0f172a`（slate-900） |
| **次文本色** | `#64748b`（slate-500） |
| **圆角** | `24px`（大卡片） / `16px`（中卡片） / `12px`（按钮） |
| **阴影** | 三层柔和阴影系统（详见设计原型文档） |
| **毛玻璃** | `backdrop-blur(20px)` + `rgba(255,255,255,0.7)` |
| **字体系列** | Inter（界面） / Noto Sans SC（汉字） / JetBrains Mono（代码） |

### 3.3 响应式设计

| 设备类型 | 断点 | 布局策略 |
|---------|------|---------|
| **移动端** | < 768px | 单列布局，底部导航栏替代顶部导航，触控目标 ≥ 44px |
| **平板** | 768px - 1024px | 双列布局，顶部导航 + 移动端菜单 |
| **桌面端** | ≥ 1024px | 三列布局，完整顶部导航栏 |

## 4. 无障碍 (A11y) 核心规范

### 4.1 ARIA 属性

| 组件 | ARIA 规范 |
|-----|----------|
| **导航按钮** | `aria-label` / `aria-expanded` / `aria-haspopup` |
| **下拉菜单** | `role="listbox"` / `role="option"` / `aria-selected` |
| **对话框** | `role="dialog"` / `aria-modal="true"` / `aria-labelledby` |
| **状态切换** | `aria-pressed` / `aria-checked` / `aria-live` |
| **图标按钮** | 必须提供 `aria-label` 文本替代 |

### 4.2 动画与动效

| 特性 | 规范 |
|-----|------|
| **尊重系统偏好** | 检测 `prefers-reduced-motion: reduce`，禁用非必要动画 |
| **动画时长** | 默认 200ms - 400ms，使用 `cubic-bezier(0.4, 0, 0.2, 1)` 缓动 |
| **焦点管理** | 所有交互元素必须可见焦点环（`ring-2 ring-emerald-500`） |
| **Skip Link** | 页面顶部提供「跳转至主内容」链接，Tab 首次聚焦时可见 |

### 4.3 键盘导航

| 按键 | 行为 |
|-----|------|
| **Tab** | 按逻辑顺序在交互元素间移动 |
| **Escape** | 关闭弹窗、菜单、对话框 |
| **Arrow Up/Down** | 在列表项间移动 |
| **Enter/Space** | 激活按钮或选择项 |

## 5. 项目目标与范围

### 5.1 项目目标

- 提供个性化、智能化的汉字学习体验
- 结合传统学习方法与现代 AI 技术
- 打造符合 Apple 设计语言的优雅用户界面
- 支持多语言国际化，覆盖全球学习者
- 实现高效的汉字书写练习和分析功能

### 5.2 目标用户

- 中文学习者（初学者到高级学习者）
- 需要练习汉字书写的用户
- 对汉字文化和词源感兴趣的用户
- 追求优质 UI/UX 体验的用户

### 5.3 项目范围

**包含功能：**
- AI 驱动的手写分析和实时反馈
- 汉字学习和练习功能
- Apple 风格深色/浅色模式支持，毛玻璃效果
- 多语言国际化（11 种语言）
- 响应式设计，支持移动到桌面全设备

**暂不包含：**
- 用户账户和进度云端同步（仅本地存储）
- 社交功能
- 在线课程购买

## 6. 核心功能

| 功能模块 | 说明 |
|---------|------|
| AI 驱动的洞察 | 对用户的手写提供实时反馈，包括笔画顺序、平衡性和美学评估 |
| 自适应学习 | 根据用户进度调整个性化学习路径 |
| 词源与文化 | 发现汉字背后的历史故事和文化背景 |
| 深色/浅色模式 | 支持持久化主题偏好和系统自动检测，毛玻璃效果自适应 |
| 国际化 (i18n) | 支持 11 种语言，包括英语、简体中文、繁体中文、西班牙语、阿拉伯语、法语、葡萄牙语、德语、日语、韩语和俄语 |
| 语言切换器 | 便捷的语言切换并支持本地存储偏好 |
| 中文优化显示 | 集成 Noto Sans SC 字体提供优雅的汉字显示 |
