设计规格说明

响应式断点

  • xs: 475px - 超小屏幕
  • sm: 640px - 小屏幕
  • md: 768px - 中等屏幕
  • lg: 1024px - 大屏幕
  • xl: 1280px - 超大屏幕

颜色系统

  • 主色: Emerald (emerald-500: #10b981)
  • 辅助色: Cyan, Blue, Purple
  • 中性色: Slate 系列
  • 深色模式: dark: 前缀

字体系统

  • 界面字体: Inter
  • 汉字字体: Noto Sans SC
  • 代码字体: JetBrains Mono
  • 汉字类: .hanzi-font

间距规范

  • 页面容器: max-w-7xl mx-auto px-4 sm:px-6
  • 卡片间距: gap-4 sm:gap-6 lg:gap-8
  • 内边距: p-4 sm:p-6 lg:p-8
  • 安全区域: safe-bottom (底部)

动画效果

  • 页面过渡: page-transition (淡入)
  • 缩放进入: animate-scale-in
  • 悬停效果: hover:-translate-y-1
  • 点击反馈: active:scale-95

国际化 (i18n)

  • 支持语言: 11种
  • 翻译函数: t('key.path')
  • 存储键: hanzi-master-locale
  • 自动检测: 浏览器语言

主题系统

  • 模式: light / dark / system
  • 存储键: hanzi-master-theme
  • Provider: ThemeProvider
  • 切换组件: ThemeToggleClient

无障碍 (A11y)

  • 语义化: header, main, nav, footer
  • ARIA: aria-expanded, aria-selected
  • 键盘导航: Tab, Enter, Space, Escape
  • 跳转链接: Skip to content

组件清单

页面组件 (src/app/)

  • page.tsx - 首页
  • layout.tsx - 根布局
  • learn/page.tsx - 学习页
  • practice/page.tsx - 练习页
  • globals.css - 全局样式

可复用组件 (src/components/)

  • locale-provider.tsx - 国际化上下文
  • locale-toggle.tsx - 语言切换
  • theme-provider.tsx - 主题上下文
  • theme-toggle.tsx - 主题切换
  • mobile-nav.tsx - 移动端导航
  • nav-link.tsx - 导航链接
  • feature-card.tsx - 功能卡片
  • stats-card.tsx - 统计卡片

i18n 模块 (src/lib/i18n/)

  • index.ts - 配置导出
  • translations/en.ts - 英语
  • translations/zh-CN.ts - 简体中文
  • translations/zh-TW.ts - 繁体中文
  • translations/es.ts - 西班牙语
  • translations/ar.ts - 阿拉伯语
  • + 6种其他语言

配置文件

  • package.json - 项目配置
  • tsconfig.json - TypeScript 配置
  • next.config.js - Next.js 配置
  • tailwind.config.ts - Tailwind 配置
  • eslint.config.js - ESLint 配置
  • vercel.json - 部署配置
