## 3. 代码问题

### 3.1 原生浏览器 API 违规 (严重)

项目规则明确禁止使用 `alert()` 或 `confirm()`，必须使用全局 `Toast` 组件。

| 文件 | 问题 |
|------|------|
| `app/components/settings/SettingsHazardZone.tsx` | 使用 `window.confirm()` |
| `app/components/ChallengeModal.tsx` | 使用 `alert()` (3 处) |

**违规代码示例**:
```tsx
// SettingsHazardZone.tsx
if (window.confirm(labels.resetConfirm)) { ... }

// ChallengeModal.tsx
alert(labels.timeUp || 'Time is up!');
alert('Failed to load character data.');
alert(`Challenge Complete! Score: ${score}`);
```

### 3.2 SSR 兼容性问题 (中等)

以下文件在服务端渲染时直接使用 `window.location.origin`:

| 文件 | 状态 |
|------|------|
| `app/components/analysis/IdiomDisplay.tsx` | ❌ 直接使用 |

```tsx
const shareUrl = `${window.location.origin}?char=...`;  // SSR 报错
```
