# 测试规范

## 测试框架

项目使用 Angular 内置的测试工具：

- Karma：测试运行器
- Jasmine：测试框架

## 测试类型

1. **单元测试**：测试独立的组件、服务和函数
2. **集成测试**：测试组件间的交互
3. **端到端测试**：测试完整的用户流程

## 测试命令

```bash
# 运行测试
npm test

# 运行测试并监听文件变化
npm test -- --watch
```

## 测试规范

1. 组件应创建对应的 `.spec.ts` 测试文件
2. 测试文件应与源代码放在同一目录下
3. 使用 `describe` 和 `it` 组织测试用例
4. 测试用例应清晰描述预期行为
5. 保持测试独立，不依赖其他测试的执行顺序
