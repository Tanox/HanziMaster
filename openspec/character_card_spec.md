# 汉字信息卡片设计规范 (Character Card Design Spec)

## 1. 概述 (Overview)
汉字信息卡片是应用的核心组件，用于展示单个汉字的详细信息、发音、释义及动态笔顺。该组件旨在提供直观、沉浸式的学习体验，帮助用户掌握汉字的书写与含义。

## 2. 界面布局 (Interface Layout)

### 2.1 整体结构
卡片采用垂直流式布局（Mobile-First），在宽屏设备上可调整为左右分栏布局。

- **顶部区域 (Header)**: 汉字主体展示与发音。
- **中部区域 (Stroke Demo)**: 动态笔顺演示区域。
- **底部区域 (Details)**: 详细信息（释义、部首、结构等）。

### 2.2 布局示意 (Wireframe)

```
+-----------------------------------+
|  [拼音 Pinyin]      [发音图标 🔊]  |
|                                   |
|          汉字 (Large Char)         |
|                                   |
+-----------------------------------+
|                                   |
|       [ 动态笔顺演示区域 ]         |
|       [ Canvas / Animation ]      |
|                                   |
|    [⏪] [⏯] [⏩] (控制栏)         |
+-----------------------------------+
|                                   |
|  基本释义: ...                     |
|  部首: ...      结构: ...          |
|  笔画数: ...                       |
|                                   |
+-----------------------------------+
```

## 3. 内容展示要求 (Content Requirements)

### 3.1 汉字主体 (Character)
- **字体**: 使用楷体或标准黑体，确保笔画清晰。
- **大小**: 醒目展示，建议字号 `6rem` (96px) 以上。
- **背景**: 可选田字格或米字格背景，辅助结构认知。

### 3.2 拼音与发音 (Pinyin & Audio)
- **拼音**: 显示在汉字上方或旁侧，标注声调。
- **音频**: 提供普通话发音播放按钮。
  - 图标: 喇叭图标 (Speaker Icon)。
  - 交互: 点击播放，播放时图标高亮或有声波动画。

### 3.3 动态笔顺 (Dynamic Stroke Order)
- **演示**: 按照标准笔顺逐笔书写。
- **颜色**:
  - **当前笔画**: 高亮色（如红色或蓝色）。
  - **已写笔画**: 深色（如黑色）。
  - **未写笔画**: 浅灰色轮廓（Ghost）。
- **控制**:
  - **播放/暂停**: 控制动画启停。
  - **重置**: 重新开始演示。
  - **速度**: 可调节播放速度 (0.5x, 1x, 2x)。

### 3.4 详细信息 (Details)
- **基本释义 (Meaning)**: 简洁明了的英文/本地化释义。
- **部首 (Radical)**: 显示部首字符及其拼音/含义。
- **结构 (Structure)**:
  - 文本描述: 如 "左右结构" (Left-Right), "上下结构" (Top-Bottom)。
  - 可视化图标 (可选): 使用简单图示表示结构（如 `⿰`, `⿱`）。
- **笔画数 (Stroke Count)**: 总笔画数。

## 4. 交互细节 (Interaction Details)

### 4.1 动画交互
- **自动播放**: 进入卡片时可配置是否自动播放笔顺。
- **点击汉字**: 点击汉字区域可触发发音播放或重新播放动画。
- **手势**: 支持左右滑动切换上/下一个汉字（如果在列表中）。

### 4.2 响应式适配
- **移动端**: 垂直堆叠，笔顺区域占据屏幕宽度。
- **桌面端**:
  - 左侧: 汉字与笔顺演示。
  - 右侧: 详细信息与扩展内容（如词组、例句）。

### 4.3 视觉反馈
- **加载状态**: 数据加载时显示骨架屏 (Skeleton)。
- **错误状态**: 无法加载汉字数据时显示友好提示。

## 5. 技术实现建议 (Implementation Notes)
- **库**: 使用 `hanzi-writer` 或类似库实现 SVG 笔顺动画。
- **数据源**: 使用 `hanzi-writer-data` 或 `MakeMeAHanzi` 数据集。
- **音频**: 使用 Web Speech API (TTS) 或预录制的 MP3 资源。
