# 原型图设计文档

## 1. 设计概述

本项目采用现代化、简洁优雅的设计风格，结合了柔和的配色方案和流畅的动画效果，为用户提供愉悦的汉字学习体验。

### 1.1 设计理念

- **简洁优雅**：采用简洁的界面设计，突出内容的重要性
- **现代美感**：使用渐变色彩、阴影效果和流畅动画
- **响应式设计**：完美适配各种屏幕尺寸
- **无障碍支持**：深色/浅色/系统主题，支持多种语言

### 1.2 配色方案

| 颜色用途 | 颜色代码 | 说明 |
|---------|---------|------|
| 主色调 | `#10b981` (emerald-500) | 用于主要按钮和强调元素 |
| 主色调深色 | `#059669` (emerald-600) | 用于按钮悬停状态 |
| 辅助色 | `#06b6d4` (cyan-500) | 用于次要强调和装饰 |
| 背景色 | `#ffffff` (light) / `#0f172a` (dark) | 浅色/深色主题背景 |
| 文本色 | `#0f172a` (light) / `#ffffff` (dark) | 主要文本颜色 |
| 次要文本 | `#475569` (light) / `#94a3b8` (dark) | 次要文本和辅助信息 |

#### 2.1.2 关键元素

**视觉演示卡片**：
- 汉字展示：大字显示 "永" 字
- 进度条：展示笔画掌握进度 (68%)
- 动画效果：悬停时轻微旋转

**功能特性区域**：
- 三个功能卡片，分别展示：
  - AI 洞察 (蓝色)
  - 词源文化 (琥珀色)
  - 自适应学习 (紫色)

## 3. 组件设计

### 3.1 导航栏 (Navbar)

**设计特点**：
- 固定顶部导航
- 透明背景，滚动时变化
- 响应式布局，移动端显示汉堡菜单
- 主题和语言切换按钮

**布局**：
```
[Logo] [Learn] [Practice]          [Theme Toggle] [Language Toggle]
```

### 3.2 功能卡片 (FeatureCard)

**设计特点**：
- 渐变背景色
- 圆角卡片
- 图标 + 标题 + 描述
- 悬停动画效果

**结构**：
```
┌─────────────────────┐
│  [Icon]             │
│  Title              │
│  Description        │
└─────────────────────┘
```

### 3.3 统计卡片 (StatsCard)

**设计特点**：
- 简洁的统计展示
- 数字强调显示
- 支持深色/浅色主题

## 4. 动画与交互

### 4.1 页面动画

| 动画名称 | 效果 | 使用场景 |
|---------|------|---------|
| `fade-in` | 淡入效果 | 页面加载时的元素显示 |
| `slide-up` | 上滑动画 | Hero 区域内容 |
| `ping` | 脉冲动画 | 状态徽章指示点 |
| `hover` | 悬停效果 | 按钮和卡片 |

### 4.2 交互反馈

- 按钮点击：轻微缩放 `active:scale-95`
- 卡片悬停：上移 + 阴影增强
- 主题切换：平滑过渡

## 5. 响应式断点

| 断点 | 屏幕宽度 | 设计调整 |
|-----|---------|---------|
| `sm` | 640px+ | 移动端优化布局 |
| `md` | 768px+ | 平板适配 |
| `lg` | 1024px+ | 桌面端完整布局 |
| `xl` | 1280px+ | 大屏幕优化 |
