# HanziMaster 设计规范文档 v4.0

## 1. 设计概述

### 1.1 设计理念

- **简洁优雅**：采用 Apple 风格的极简界面设计，突出汉字内容的重要性
- **现代美感**：使用 OKLCH 语义化颜色、柔和阴影和流畅动画
- **响应式设计**：完美适配桌面端、平板和移动端
- **无障碍支持**：支持深色/浅色/系统主题，11 种语言，键盘导航

### 1.2 设计原则

1. **内容优先**：汉字是界面的主角，UI 元素不应抢夺注意力
2. **克制用色**：以黑白灰为主，Apple Blue 作为唯一强调色
3. **呼吸感**：大量留白，元素间距遵循 8px 网格系统
4. **一致性**：所有组件遵循统一的设计语言

## 3. 组件库规范

### 3.1 基础组件

#### 3.1.1 Button 按钮

**变体 (Variants)**:
- `default`: 主按钮，黑底白字 (深色模式反转)
- `outline`: 边框按钮，透明背景
- `ghost`: 幽灵按钮，仅文字+悬停背景
- `destructive`: 危险操作，红色系
- `link`: 链接样式按钮

**尺寸 (Sizes)**:
- `xs`: 24px 高
- `sm`: 28px 高
- `default`: 32px 高
- `lg`: 36px 高

**状态 (States)**:
- Normal: 默认状态
- Hover: `scale(1.02)` + 阴影增强
- Active: `scale(0.98)` + `translateY(1px)`
- Disabled: `opacity: 0.4` + `pointer-events: none`
- Focus: `ring` 焦点环

#### 3.1.2 Input 输入框

**状态**:
- Default: 1px border `#e8e8ed`
- Focus: border `#007aff` + `box-shadow: 0 0 0 4px rgba(0,122,255,0.1)`
- Error: border `#ff3b30` + `box-shadow: 0 0 0 4px rgba(255,59,48,0.1)`
- Success: border `#34c759` + `box-shadow: 0 0 0 4px rgba(52,199,89,0.1)`
- Disabled: `opacity: 0.5`

#### 3.1.3 Badge 徽章

**变体**:
- Default: Blue 背景 8% 透明度
- Success: Green 背景 8% 透明度
- Warning: Orange 背景 8% 透明度
- Purple: Purple 背景 8% 透明度

### 3.2 复合组件

#### 3.2.1 Card 卡片

**结构**:
```
┌─────────────────────┐
│  Header (optional)  │
├─────────────────────┤
│                     │
│      Content        │
│                     │
├─────────────────────┤
│  Footer (optional)  │
└─────────────────────┘
```

**样式**:
- 背景: `var(--card)`
- 圆角: 20px-32px
- 边框: 1px `var(--border)`
- 阴影: 无 (默认) → `shadow-lg` (悬停)
- 过渡: `transform 0.3s`, `box-shadow 0.3s`

#### 3.2.2 Dialog 对话框

**结构**:
- 遮罩: `bg-black/40` + `backdrop-blur-sm`
- 容器: 圆角 24px, 最大宽度 520px-600px
- 动画: `scale-in` + `fade-in`
- 关闭: 点击遮罩 / ESC 键 / 关闭按钮

#### 3.2.3 Sheet 抽屉

**结构**:
- 遮罩: `bg-black/40` + `backdrop-blur-sm`
- 容器: 右侧滑入, 宽度 280px, 最大 85vw
- 动画: `slide-in-right`

### 3.3 业务组件

#### 3.3.1 FeatureCard 功能卡片

**结构**:
- 图标容器: 80px-96px, 渐变背景, 圆角 20px
- 标题: 20px-26px, Semibold
- 描述: 15px-16px, Regular, 灰色

**交互**:
- Hover: `translateY(-8px)` + `shadow-lg` + 图标 `scale(1.15)`
- 过渡: `0.35s cubic-bezier(0.4, 0, 0.2, 1)`

#### 3.3.2 StatsCard 统计卡片

**结构**:
- 数值: 40px-44px, Bold, 品牌色
- 标签: 15px, Medium, 灰色
- 顶部装饰线: 3px 高, 品牌色

**交互**:
- Hover: `translateY(-4px)` + `shadow-md`

#### 3.3.3 CharacterCard 汉字卡片

**结构**:
- 汉字: 48px-56px, Light 字重
- 拼音: 13px, Medium, 灰色
- 宽高比: 1:1

**交互**:
- Hover: `scale(1.03)` + border `#007aff` + `shadow-md`
- Selected: 深色背景 + 白色文字 + `shadow-glow`
- 过渡: `0.3s cubic-bezier(0.4, 0, 0.2, 1)`

#### 3.3.4 EmptyState 空状态

**结构**:
- 图标容器: 64px, 灰色背景圆形
- 标题: 18px, Semibold
- 描述: 15px, 灰色, 最大宽度 280px
- 操作按钮: 主要或次要按钮 (可选)

**使用场景**:
- 无学习记录
- 无搜索结果
- 加载失败
- 无网络连接

#### 3.3.5 Toast 通知

**类型**:
- `success`: 绿色背景, 勾选图标, 3s 自动消失
- `error`: 红色背景, X 图标, 5s 自动消失或手动关闭
- `warning`: 琥珀色背景, 警告图标, 4s 自动消失
- `info`: 蓝色背景, 信息图标, 3s 自动消失

**行为**:
- 位置: 固定右上角, `z-index: 100`
- 动画: 右侧滑入 (`slide-in-right`), 淡出消失
- 堆叠: 多个 Toast 垂直排列, 间距 8px
- 关闭: 点击 X 按钮或自动超时

#### 3.3.6 Skeleton 骨架屏

**变体**:
- `default`: 通用矩形
- `card`: 圆角 20px, 用于卡片占位
- `text`: 用于文本行占位
- `avatar`/`circle`: 圆形, 用于头像/图标占位

**预置模式**:
- `SkeletonCard`: 图标 + 标题 + 描述行
- `SkeletonCharacterGrid`: 汉字网格占位
- `SkeletonStats`: 统计卡片占位

**动画**: `animate-pulse`, 灰色背景

#### 3.3.7 Loading 加载

**变体**:
- `Loading`: 旋转指示器 + 可选文字
- `LoadingOverlay`: 遮罩层 + 居中加载指示器
- `LoadingPage`: 页面级加载, 最小高度 60vh

**尺寸**:
- `sm`: 20px
- `md`: 32px (默认)
- `lg`: 48px

## 5. 交互标准

### 5.1 交互模式

#### 5.1.1 页面导航

- 页面切换: `fade-in-up` 动画, 0.5s
- 滚动行为: `scroll-behavior: smooth`
- 当前页面指示: 导航项底部圆点

#### 5.1.2 按钮交互

- Hover: `scale(1.02)` + 阴影增强, 0.2s
- Active: `scale(0.98)` + `translateY(1px)`, 0.1s
- Focus: 2px outline + 2px offset, `#007aff`

#### 5.1.3 卡片交互

- Hover: `translateY(-4px to -8px)` + `shadow-lg`, 0.3s
- 选中: 边框高亮 + 光晕阴影

### 5.2 反馈机制

#### 5.2.1 加载状态

- 骨架屏: `animate-pulse`, 灰色背景
- 加载指示器: 旋转动画, Apple Blue
- 最小加载时间: 200ms (避免闪烁)

#### 5.2.2 成功状态

- Toast: 绿色背景, 顶部滑入, 3s 自动消失
- 图标: 勾选标记
- 声音: 可选成功音效

#### 5.2.3 错误状态

- Toast: 红色背景, 顶部滑入, 5s 自动消失或手动关闭
- 表单错误: 红色边框 + 错误文字
- 图标: 感叹号或 X

### 5.3 错误处理

#### 5.3.1 表单验证

- 实时验证: 输入后 500ms debounce
- 错误显示: 红色边框 + 下方错误文字
- 成功显示: 绿色边框 + 下方提示文字

#### 5.3.2 网络错误

- 重试按钮: 显示在错误信息旁
- 离线提示: 顶部横幅通知

### 5.4 空状态

#### 5.4.1 空状态设计

- 图标: 64px, 灰色背景圆形
- 标题: 18px, Medium
- 描述: 15px, 灰色, 最大宽度 280px
- 操作按钮: 主要或次要按钮

#### 5.4.2 空状态类型

- 无学习记录: 📚 图标 + "开始学习" CTA
- 无搜索结果: 🔍 图标 + "清除搜索" 按钮
- 加载失败: ⚠️ 图标 + "重新加载" 按钮
- 无网络: 📡 图标 + "检查网络" 提示

| lg | 1024px+ | 桌面端完整布局 |

- 功能卡片: 3 列 → 1 列
- 统计卡片: 3 列 → 1 列

## 7. 无障碍设计

### 7.1 键盘导航

- Tab 顺序: 逻辑顺序
- Focus 可见: 2px outline, `#007aff`
- Skip link: 跳转到主要内容

### 7.2 屏幕阅读器

- ARIA 标签: 所有交互元素
- 角色: `role="listbox"`, `role="option"`, `role="dialog"`
- 状态: `aria-selected`, `aria-expanded`, `aria-hidden`

### 7.3 减少动画

```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```
