# 汉字笔顺教学应用：课程大纲与学习路径说明文档

## 1. 设计理念 (Design Philosophy)
本课程体系遵循**“字元（笔画） -> 构件（部首） -> 结构（间架） -> 语境（词汇/成语）”**的认知规律。结合应用内置的 SRS（间隔重复系统）和 AI 智能分析功能，旨在为用户提供一条科学、平滑且充满成就感的汉字学习路径。

## 2. 学习路线图 (Learning Roadmap)

针对不同水平的用户，应用将提供以下三条清晰的学习路线：

*   🌱 **L1 初学者 (Beginner)：破冰与筑基**
    *   **目标**：消除对汉字的恐惧感，掌握基本笔画和书写规则，能写出最简单的独体字。
    *   **核心**：6大基本笔画 + 7大笔顺规则 + 50个高频独体字。
*   🌿 **L2 进阶者 (Intermediate)：解构与重组**
    *   **目标**：理解汉字的“拼图”本质，掌握常见部首的含义，学习复合结构汉字。
    *   **核心**：30个高频偏旁部首 + 间架结构（左右、上下、包围） + 300个生活常用字。
*   🌳 **L3 高阶者 (Advanced)：语境与文化**
    *   **目标**：掌握形声字规律，攻克易混淆字和多笔画字，拓展成语和词汇量。
    *   **核心**：形声字辨析 + 高难度汉字 + 成语典故 + 结合 AI 探索字源（Etymology）。

## 3. 详细课程大纲 (Detailed Syllabus)

### 第一阶段：初学者 (L1) - 基础笔画与规则
*   **模块 1.1：万物之始（基本笔画）**
    *   **内容**：横（一）、竖（丨）、撇（丿）、捺（乀）、点（丶）、提（㇀）。
    *   **练习字例**：一、二、三、十、八、人、主。
*   **模块 1.2：转折的艺术（复合笔画）**
    *   **内容**：横折（ㄱ）、竖折（ㄴ）、竖钩（亅）、弯钩（㇁）、斜钩（㇂）。
    *   **练习字例**：口、日、月、小、水、我。
*   **模块 1.3：规矩成方圆（七大笔顺规则）**
    *   **内容**：从上到下、从左到右、先横后竖、先撇后捺、从外到内、先进后关、先中间后两边。
    *   **练习字例**：李、明、十、木、同、回、水。

### 第二阶段：进阶者 (L2) - 部首与结构
*   **模块 2.1：汉字的“偏旁”家族（高频部首）**
    *   **内容**：与人相关（亻、彳）、与自然相关（氵、火、木、艹）、与动作/身体相关（扌、足、月）。
    *   **练习字例**：你、海、秋、林、花、打、明。
    *   **应用结合**：利用 AI 分析面板中的“部首（Radical）”功能进行强化记忆。
*   **模块 2.2：汉字的建筑学（间架结构）**
    *   **内容**：左右结构（宽窄比例）、上下结构（高低比例）、包围结构（全包围、半包围）。
    *   **练习字例**：好、尖、国、闪、这。
*   **模块 2.3：生活词汇构建（主题字库）**
    *   **内容**：数字与时间、方向与位置、家庭称谓。
    *   **练习字例**：百、千、年、上、下、左、右、爸、妈。

### 第三阶段：高阶者 (L3) - 深度与广度
*   **模块 3.1：形声字的秘密（声旁与形旁）**
    *   **内容**：理解“形旁表意，声旁表音”的规律，实现批量识字。
    *   **练习字例**：青、请、清、情、晴、蜻。
*   **模块 3.2：火眼金睛（易混淆字辨析）**
    *   **内容**：笔画微小差异导致的字义变化。
    *   **练习字例**：己/已/巳、戊/戌/戍、买/卖。
*   **模块 3.3：挑战极限（高难度多笔画字）**
    *   **内容**：15画以上的复杂汉字书写挑战。
    *   **练习字例**：繁、藏、赢、餐、疆。
*   **模块 3.4：成语与文化（结合成语功能）**
    *   **内容**：学习由四个字组成的成语，了解背后的历史典故。
    *   **练习字例**：一马当先、画龙点睛、守株待兔。
    *   **应用结合**：直接联动应用内的“成语解析（Idiom Analysis）”面板。

## 4. 学习建议与应用功能结合 (Integration with App Features)

1.  **闯关模式 (Challenge Mode)**：将上述模块转化为应用内的“关卡”。用户必须在“练习模式 (Practice Mode)”中达到 **PROFICIENT (能品)** 或以上评分，才能解锁下一课。
2.  **智能复习 (SRS Integration)**：用户学过的基础笔画字和部首，会自动加入应用的 SRS 记忆库，在首页的 HistoryPanel 中提示到期复习。
3.  **AI 助记 (AI Mnemonics)**：对于 L2 和 L3 阶段的复杂汉字，强烈建议用户开启设置中的 `showMnemonic` 和 `showEtymology`，利用大模型生成的象形解释和记忆口诀辅助学习。
