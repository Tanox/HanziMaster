# 06. 合规、质量与运维

**版本**: v1.0.0
**职责**: 定义交付质量、隐私安全与运维流程

## 1. 质量保证 (QA Test Cases)
*   **离线韧性审计**: 验证在飞行模式下，已同步的 HSK 4 级字词解析及笔画评分逻辑是否正常。
*   **手写边界测试**: 验证在 9:16 (手机) 与 3:4 (平板) 屏幕下 Canvas 坐标映射的准确性。
*   **更新中断恢复**: 验证在下载离线词库时网络中断，重连后是否支持断点续传。

## 2. 安全与隐私
*   **BYOK 策略**: 严禁将用户的自定义 Gemini Key 上传至任何中间服务器。
*   **数据隔离**: 练习轨迹仅用于实时评分计算，计算完成后立即销毁，不存储具体轨迹坐标。
*   **合规性**: 适配各地区隐私政策（如 GDPR），支持一键“重置应用”彻底抹除本地足迹。

## 3. 运维部署 (DevOps)
*   **构建顺序**: `npm install -> copy-data -> tsc -> vite build`。
*   **PWA 策略**: Workbox `CacheFirst` 缓存 `/hanzi-data/` 目录，有效期 1 年。
*   **更新机制**: 每次部署后必须推送 SW 更新，通过 `ReloadPrompt` 强制用户刷新应用。
