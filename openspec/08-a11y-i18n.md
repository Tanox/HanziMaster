## 1. 无障碍 (A11y) 总目标

| 目标 | 说明 |
|-----|------|
| **WCAG 2.1 AA** | 所有页面达到 Web Content Accessibility Guidelines 2.1 AA 级别 |
| **键盘可用** | 所有功能可通过键盘（Tab/Enter/Escape/Arrow 键）操作 |
| **屏幕阅读器** | 与主流屏幕阅读器兼容（VoiceOver、NVDA、JAWS） |
| **动效友好** | 尊重 `prefers-reduced-motion` 用户偏好 |
| **色彩对比** | 正文文本与背景对比度 ≥ 4.5:1 |

## 2. ARIA 属性规范

### 2.1 导航组件 ARIA

| 组件 | ARIA 实现 |
|-----|----------|
| **顶部导航 `<nav>`** | `aria-label="主导航"` |
| **NavLink（活跃态）** | `aria-current="page"` |
| **底部移动导航** | `aria-label="页面导航"` |
| **移动端菜单按钮** | `aria-label="打开菜单"` + `aria-expanded="true/false"` + `aria-controls="mobile-menu"` |
| **MobileNav 抽屉** | `role="dialog"` + `aria-modal="true"` + `aria-labelledby="mobile-menu-title"` |

### 2.2 主题/语言切换 ARIA

| 组件 | ARIA 实现 |
|-----|----------|
| **ThemeToggle 按钮** | `aria-label="切换主题：当前浅色"`（动态文本）+ `title="浅色模式"` |
| **LocaleToggle 按钮** | `aria-haspopup="listbox"` + `aria-expanded="true/false"` + `aria-label="选择语言"` |
| **语言下拉菜单容器** | `role="listbox"` + `aria-label="语言列表"` |
| **语言菜单项** | `role="option"` + `aria-selected="true/false"` + `tabIndex={选中项为 0，其余 -1}` |

### 2.3 按钮与交互元素 ARIA

| 元素 | ARIA 实现 |
|-----|----------|
| **纯图标按钮** | 必须包含 `aria-label`（如 `aria-label="切换主题"`） |
| **主要 CTA 按钮** | 自然按钮元素 `<button>`，无需额外 role |
| **动态内容区域** | `aria-live="polite"`（如 AI 反馈面板） |
| **Skip Link** | `<a href="#main-content" aria-label="跳转至主内容">` |

### 2.4 常见 ARIA 错误避免

```tsx
// ✅ 正确：原生按钮 + aria-label
<button aria-label="切换主题" className="...">☀️</button>

// ❌ 错误：div 代替 button（缺少键盘和语义）
<div onClick={toggle}>☀️</div>

// ✅ 正确：role + tabindex + onKeyDown 处理
<div role="button" tabIndex={0} onKeyDown={handleKeyDown} onClick={toggle}>☀️</div>
```

## 3. 键盘导航规范

### 3.1 全局键盘行为

| 按键 | 全局行为 |
|-----|---------|
| **Tab** | 按视觉顺序在交互元素间移动焦点，焦点在 `focus:ring-2 focus:ring-emerald-500` 可见 |
| **Shift + Tab** | 反向移动焦点 |
| **Escape** | 关闭所有弹出层（下拉菜单、抽屉、对话框），焦点回到触发器 |
| **Enter / Space** | 激活当前聚焦的按钮或链接 |

### 3.2 下拉菜单键盘行为（语言切换）

| 按键 | 行为 |
|-----|------|
| **Enter/Space（触发器）** | 打开菜单，焦点移至当前选中项 |
| **Arrow Down** | 移动到下一项，循环 |
| **Arrow Up** | 移动到上一项，循环 |
| **Enter（菜单项）** | 选择当前项，关闭菜单，焦点回到触发器 |
| **Escape** | 关闭菜单，焦点回到触发器 |
| **点击外部** | 关闭菜单 |

### 3.3 抽屉（MobileNav）键盘行为

| 按键 | 行为 |
|-----|------|
| **打开时** | 焦点自动移到抽屉的第一个可交互元素 |
| **Tab** | 在抽屉内循环（焦点陷阱） |
| **Escape** | 关闭抽屉，焦点回到触发器 |
| **打开时** | body 滚动锁定（`document.body.style.overflow = 'hidden'`） |

### 3.4 焦点顺序 (Tab Order)

```
首页焦点顺序：
  1. [Skip Link] 跳转至主内容（视觉隐藏，Tab 首次聚焦显示）
  2. Logo
  3. NavLink: Learn
  4. NavLink: Practice
  5. ThemeToggle（主题切换）
  6. LocaleToggle（语言切换）
  7. CTA: 开始学习
  8. CTA: 探索字库
  9. (移动底部导航) Home / Learn / Practice 图标
  10. Footer 链接
```

## 4. Skip Link 实现

### 4.1 代码示例

```tsx
// layout.tsx
<a
  href="#main-content"
  className="
    sr-only             // 默认视觉隐藏
    focus:not-sr-only   // 聚焦时显示
    focus:fixed focus:top-4 focus:left-4 focus:z-50
    focus:rounded-xl focus:bg-emerald-500 focus:px-4 focus:py-3
    focus:text-white focus:font-medium focus:shadow-lg
  "
>
  跳转至主内容
</a>

<main id="main-content">
  {children}
</main>
```

### 4.2 Tailwind `sr-only` 类解析

```css
.sr-only {
  position: absolute;
  width: 1px; height: 1px;
  padding: 0; margin: -1px;
  overflow: hidden;
  clip: rect(0, 0, 0, 0);
  white-space: nowrap;
  border-width: 0;
}
```

> Skip Link 是满足 WCAG 2.4.1 Bypass Blocks（绕过重复块）要求的标准实现。

## 5. prefers-reduced-motion 动效降级

### 5.1 检测方法

```css
/* 全局 CSS 降级 */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    /* 保留颜色过渡，去除位移和缩放 */
    transition-property: color, background-color, border-color !important;
  }
}
```

### 5.2 React 端检测（可选）

```tsx
'use client';
import { useEffect, useState } from 'react';

export function useReducedMotion() {
  const [reduced, setReduced] = useState(false);
  useEffect(() => {
    const mq = window.matchMedia('(prefers-reduced-motion: reduce)');
    setReduced(mq.matches);
    const listener = () => setReduced(mq.matches);
    mq.addEventListener('change', listener);
    return () => mq.removeEventListener('change', listener);
  }, []);
  return reduced;
}

// 用法：className={reduced ? '' : 'hover:-translate-y-1'}
```

### 5.3 禁用的动画清单

| 动画 | 禁用条件 |
|-----|---------|
| `hover:-translate-y-1` 卡片浮起 | reduce-motion 时禁用 |
| `hover:scale-105` 按钮缩放 | reduce-motion 时禁用 |
| `animate-pulse` 脉冲点 | reduce-motion 时禁用 |
| `transition-all duration-300` 全过渡 | 降级为仅颜色过渡 |
| 渐变色文字动画 | reduce-motion 时降级为纯色 |

## 6. 色彩与对比度

### 6.1 色彩对比检查表

| 前景色 | 背景色 | 对比度 | WCAG AA (4.5:1) | 用途 |
|-------|-------|--------|----------------|------|
| `slate-900` | `white` | 15.6 : 1 | ✅ 通过 | 主文本（浅色模式） |
| `slate-500` | `white` | 7.5 : 1 | ✅ 通过 | 次文本（浅色模式） |
| `white` | `slate-900` | 15.6 : 1 | ✅ 通过 | 主文本（深色模式） |
| `slate-300` | `slate-900` | 10.4 : 1 | ✅ 通过 | 次文本（深色模式） |
| `white` | `emerald-500` | 3.3 : 1 | ✅ 大号文本 | CTA 按钮文字 |
| `emerald-600` | `white` | 3.5 : 1 | ✅ 大号文本 | 链接文字 |
| `emerald-900` | `emerald-100` | 13.2 : 1 | ✅ 通过 | 徽章文字 |

### 6.2 避免仅用颜色传达信息

```tsx
// ✅ 正确：颜色 + 文字/图标
<span className="inline-flex items-center gap-2">
  <span className="w-2 h-2 rounded-full bg-emerald-500 animate-pulse" />
  AI-Powered Learning
</span>

// ❌ 错误：仅用颜色区分
<div className="text-green-500">已学习</div>
```

### 6.3 焦点环规范

```tsx
// 全局焦点样式（globals.css）
:focus-visible {
  outline: 2px solid #10b981;
  outline-offset: 2px;
  border-radius: 12px;
}

// 或在按钮上使用 Tailwind：
<button className="focus:outline-none focus:ring-2 focus:ring-emerald-500 focus:ring-offset-2">
```

## 9. 语言切换持久化

### 9.1 流程图

```
用户选择语言
    │
    ▼
LocaleProvider.setLocale('zh-CN')
    │
    ├── 更新 React State（触发重渲染）
    │
    ├── 更新 document.documentElement.lang = 'zh-CN'
    │       ↳ 供屏幕阅读器和浏览器识别
    │
    ├── 更新 document.documentElement.dir='rtl' (仅阿拉伯语)
    │       ↳ 支持从右到左排版
    │
    └── 保存到 localStorage['hanzi-master-locale']
            ↳ 下次启动时自动恢复

首次加载 → 读取 localStorage → 匹配成功？→ 使用该语言
                      ↓ 未匹配
                  检测 navigator.language
                      ↓ 匹配到支持的语言？
                      ├── 是 → 使用该语言
                      └── 否 → 回退到 'en'
```

### 9.2 RTL 支持（阿拉伯语）

```css
/* globals.css - 自动处理 */
html[dir="rtl"] {
  /* Tailwind rtl: 前缀可用于微调 */
}
```

```tsx
// LocaleProvider 内部
const rtlLocales = ['ar'];
useEffect(() => {
  document.documentElement.lang = locale;
  document.documentElement.dir = rtlLocales.includes(locale) ? 'rtl' : 'ltr';
}, [locale]);
```

## 10. 日期与数字本地化

```tsx
// 工具函数
export function formatDate(date: Date, locale: string): string {
  return new Intl.DateTimeFormat(locale, {
    year: 'numeric', month: 'long', day: 'numeric',
  }).format(date);
}

export function formatNumber(n: number, locale: string): string {
  return new Intl.NumberFormat(locale).format(n);
}

// 示例（学习页统计）
const { locale } = useTranslation();
<span>{formatNumber(128, locale)} {t('practice.totalMastered')}</span>
```
