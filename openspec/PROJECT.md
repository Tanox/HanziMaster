# 项目说明

## 项目概述

HanziMaster 汉字大师是一个现代化的、基于人工智能的汉字学习平台。它利用先进的AI技术提供关于笔画顺序、平衡和美感的个性化反馈，帮助用户高效学习汉字。

## 项目基本信息

| 项目属性 | 值 |
|---------|-----|
| 项目名称 | HanziMaster 汉字大师 |
| 当前版本 | v2.2.0 |
| 主要语言 | TypeScript |
| 项目类型 | Web 应用 |
| 前端框架 | Next.js 15 + React 19 |
| Node.js 要求 | >= 24.5.0 |

## 核心功能

1. **AI 驱动的洞察**：对用户的手写提供实时反馈，包括笔画顺序、平衡性和美学评估
2. **自适应学习**：根据用户进度调整个性化学习路径
3. **词源与文化**：发现汉字背后的历史故事和文化背景
4. **深色/浅色模式**：支持持久化主题偏好和系统自动检测
5. **国际化支持**：支持 11 种语言（英、中、日、韩、西、法、德、葡、俄、阿等），自动检测浏览器语言
6. **中文优化显示**：集成 Noto Sans SC 字体提供优雅的汉字显示

## 目标用户

- 中文学习者（初学者到高级学习者）
- 需要练习汉字书写的用户
- 对汉字文化和词源感兴趣的用户

## 项目价值

- 提供个性化、智能化的汉字学习体验
- 结合传统学习方法与现代 AI 技术
- 美观、响应式的用户界面
