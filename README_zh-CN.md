## 📖 简介

**汉字大师** 是一款**离线优先**的 PWA 应用。我们遵循 **“脚手架 (Scaffolding)”** 教学法，构建了 **“观察 -> 临摹 -> 默写”** 的学习闭环。

## 📚 项目规范全书 (OpenSpec)

本项目严格遵循 **OpenSpec** 标准，文档即真理：

1.  [**00. 项目总纲与协作策略**](./openspec/00_STRATEGY.md) - 愿景、AI Agent 协作约束。
2.  [**01. 产品定义与用户旅程**](./openspec/01_PRODUCT_UX.md) - 需求、Alex 画像、营销策略。
3.  [**02. 技术架构与工程准则**](./openspec/02_ARCHITECTURE.md) - Hook 分层、代码风格、开发指南。
4.  [**03. 设计系统与感知体验**](./openspec/03_DESIGN_SYSTEM.md) - 纸墨美学、A11y、国际化。
5.  [**04. 逻辑引擎与数据协议**](./openspec/04_LOGIC_DATA.md) - 评分算法、API 协议、数据模型。
6.  [**05. 教学大纲与学习算法**](./openspec/05_PEDAGOGY.md) - HSK 体系、SRS 复习算法。

