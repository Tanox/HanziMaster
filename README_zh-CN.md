## 📖 简介

**汉字大师** 是一款**离线优先**的 PWA 应用。我们遵循 **“脚手架 (Scaffolding)”** 教学法，构建了“观察 -> 临摹 -> 默写”的学习闭环，旨在通过逐步减少辅助来巩固用户的肌肉记忆和结构理解。
